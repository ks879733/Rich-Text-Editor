# Rich Text Editor

A simple Rich Text Editor built using HTML, CSS, and JavaScript.

## Features

- Bold, Italic, and Underline text formatting
- Text Alignment (Left, Center, Right, Justify)
- Add and Edit Links
- Change Text Color
- Change Background Color
- Save Content
- Clear Editor Content
- User-friendly Interface
- Real-time Text Editing

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Open the editor in your browser.
2. Type your content.
3. Use the toolbar to format text.
4. Save or clear content as needed.
